# Trigger Keywords

## Primary Triggers

- find meme coin
- trending meme
- hot meme today
- smart money buy
- low cap meme
- meme under 10M market cap
- active Solana meme
- meme with high volume
- meme with holder growth
- scan meme opportunities

## Natural Language Triggers

- find me a good meme today
- what meme is trending now
- show smart money meme plays
- find meme with breakout potential
- discover hot Solana meme coins
- show meme opportunities with active buyers
- find low risk meme trades
- detect meme volume spikes
- scan smart money meme entries

## Required Context

Trigger only if request contains at least one of:

- meme
- token
- Solana
- smart money
- market cap
- trading
- holders
- volume
- breakout

## Avoid Triggering On

- NFT search
- NFT collections
- Bitcoin analysis
- Bitcoin price prediction
- weather requests
- coding tasks
- unrelated crypto news
- general news
- wallet troubleshooting
- blockchain development
- airdrop guides
- node setup requests
- system administration questions

## Trigger Decision Rule

Activate only when the user intent is related to:

1. Meme token discovery
2. Meme trading opportunities
3. Smart money tracking
4. Volume or holder analysis
5. Trade setup generation

Do not activate for non-trading requests.
